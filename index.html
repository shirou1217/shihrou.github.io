SuperComputer Competetion

2023 HiPAC championship

I'm the team leader

performance optimize:Basilisk、FourcastNet、HPL、HPCG

I am responsible for FourCastNet, which is built on NVIDIA's open-source physics-based machine learning platform, Modulus. FourCastNet leverages data-driven machine learning to train its predictive models, marking a significant advancement in atmospheric applications. It represents a cutting-edge fusion of computational fluid dynamics (CFD) and multi-node computing. In the contest, we need to train the model with the provided dataset and draw wind speed and pressure charts for the specified dates.

Learn more about my research on FourCastNet.

2023 HPCAI third place

I'm the team leader

performance optimize: MPAS、BLOOMS

I am responsible for BLOOM, an autoregressive Large Language Model (LLM) trained to continue text from a prompt using vast amounts of text data and industrial-scale computational resources. With its 176 billion parameters, BLOOM can generate text in 46 natural languages and 13 programming languages. BLOOM can also be instructed to perform text tasks it hasn't been explicitly trained for by casting them as text generation tasks. In the contest, we need to run the benchmark inference of BLOOM and optimize its throughput.

Learn more about my research on BLOOM.
